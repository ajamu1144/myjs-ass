History of JavaScript

1995

JavaScript Created

Brendan Eich created JavaScript while working at Netscape Communications Corporation. Initially named Mocha, it was later renamed to JavaScript.

1996

JavaScript 1.0 Released

JavaScript 1.0 was officially released as part of Netscape Navigator 2.0.

1997

ECMAScript Standard

The ECMAScript standard was first published by ECMA International, providing a specification for JavaScript.

1999

ECMAScript 3

ECMAScript 3 was released, introducing regular expressions, try/catch exception handling, and other features.

2005

AJAX and Web 2.0

The introduction of AJAX led to a surge in JavaScript usage, enabling asynchronous data loading and interactive web applications.

2009

ECMAScript 5

ECMAScript 5 introduced new features such as JSON support, strict mode, and accessor properties.

2015

ECMAScript 6 (ES6)

ES6, also known as ECMAScript 2015, introduced major new features such as classes, modules, and arrow functions.

2016 - Present

Ongoing Updates

JavaScript continues to evolve with annual updates to the ECMAScript specification, adding new features and improving the language.

Facts About ES6

  • Arrow Functions: Shorter syntax for writing functions.
  • Classes: Introduced a more concise way to create objects and handle inheritance.
  • Template Literals: Allow embedded expressions and multi-line strings.
  • Destructuring: Extract values from arrays or properties from objects into distinct variables.
  • Enhanced Object Literals: Improved syntax for defining methods and properties in objects.
  • Promises: Provide a way to handle asynchronous operations more cleanly.
  • Modules: Introduced import and export statements for modularizing code.
  • Let and Const: New ways to declare variables with block scope.
