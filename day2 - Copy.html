JavaScript Variable Declarations

Keyword Scope Hoisting Re-assignment Special Notes
var Function or global Hoisted and initialized with undefined Can be re-assigned Can lead to scope issues
let Block Hoisted but not initialized Can be re-assigned Has block scope
const Block Hoisted but not initialized Cannot be re-assigned Contents of objects/arrays can be modified
